# BlazquezCañasLaura_M17UF1R1
VVVVVV Game

## Instruccions de joc
### Inputs
- Moviment lateral esquerre: A o <--  
- Moviment lateral dret: D o -->  
- Canvi de gravetat: Espai  
- Menú pausa: Escape  

### Descripció
- Génere: Plataformes  
- Tipus: Scroll Horitzontal
